# Airport Management System

A comprehensive desktop application for managing airport operations, built using Java and following Object-Oriented Programming principles.

## Features

- Flight Management
- Passenger Management
- Ticket Booking System
- Airport Staff Management
- Real-time Flight Status Updates
- Booking Management
- User Authentication and Authorization

## Technologies Used

- Java
- Maven
- Object-Oriented Programming
- JavaFX (for GUI)

## Getting Started

### Prerequisites

- Java JDK 11 or higher
- Maven
- Your favorite IDE (IntelliJ IDEA recommended)

### Installation

1. Clone the repository
```bash
git clone https://github.com/Joe-world1357/airport-management-system.git
```

2. Navigate to the project directory
```bash
cd airport-management-system
```

3. Build the project
```bash
mvn clean install
```

4. Run the application
```bash
mvn javafx:run
```
